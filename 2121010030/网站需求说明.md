# "我的游戏星球"个人游戏展示网站 - 需求说明文档

## 一、项目概述

### 1.1 项目名称
**我的游戏星球** —— 个人游戏经历与推荐展示网站

### 1.2 项目背景
作为一名热爱游戏的玩家，我希望通过一个个人网站来展示自己玩过的游戏、分享对游戏的看法与评价，并为不同类型的游戏提供推荐。网站面向对游戏感兴趣的网友，帮助他们了解各类游戏的特点，发现适合自己的游戏。

### 1.3 项目目标
- 展示个人游戏经历与游戏评价
- 按游戏类型进行分类浏览与筛选
- 支持关键词搜索快速定位游戏
- 通过数据可视化展示个人游戏偏好
- 提供美观、交互友好的用户体验

---

## 二、页面结构

### 2.1 整体布局
网站采用单页应用（SPA）结构，包含以下主要板块：

| 板块 | 说明 |
|------|------|
| 导航栏 | 固定顶部，包含网站标题、分类快捷导航、搜索框 |
| Hero 横幅 | 首屏大图，展示网站主题与标语 |
| 游戏分类筛选区 | 标签式分类按钮，支持按类型筛选 |
| 游戏卡片展示区 | 以卡片形式展示所有游戏，包含封面图、名称、类型、评分 |
| 游戏详情弹窗 | 点击卡片弹出详情，包含完整介绍、截图、评价 |
| 数据统计可视化区 | 雷达图展示个人游戏偏好维度 |
| 页脚 | 版权信息、联系方式 |

### 2.2 页面层级
```
首页（单页）
├── 导航栏（固定顶部）
│   ├── 网站Logo/标题
│   ├── 分类快捷链接
│   └── 搜索框
├── Hero 横幅区
├── 游戏展示区
│   ├── 分类筛选标签
│   ├── 搜索结果提示
│   └── 游戏卡片网格
├── 数据统计区
│   └── 个人游戏偏好雷达图
└── 页脚
```

## 三、风格配色

### 3.1 设计风格
**明亮清新风格** —— 以浅色为基底，搭配活泼的彩色点缀，营造轻松愉快的游戏氛围。

### 3.2 配色方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主背景色 | 浅灰白 | `#F5F7FA` |
| 卡片背景色 | 纯白 | `#FFFFFF` |
| 主强调色 | 天蓝色 | `#4FC3F7` |
| 辅助强调色 | 珊瑚橙 | `#FF7043` |
| 标题文字色 | 深灰蓝 | `#2C3E50` |
| 正文文字色 | 中灰 | `#546E7A` |
| 边框/分割线 | 浅灰 | `#E0E0E0` |
| 导航栏背景 | 白色半透明 | `rgba(255,255,255,0.95)` |
| 分类标签激活 | 天蓝色 | `#4FC3F7` |
| 分类标签默认 | 浅灰 | `#ECEFF1` |
| 评分星星 | 金黄色 | `#FFC107` |

### 3.3 字体方案
- 标题字体：`'Noto Sans SC', 'Microsoft YaHei', sans-serif`
- 正文字体：`'Noto Sans SC', 'Microsoft YaHei', sans-serif`
- 英文/数字：`'Poppins', sans-serif`

### 3.4 设计元素
- 卡片圆角：`12px`
- 卡片阴影：`0 4px 15px rgba(0,0,0,0.08)`
- 按钮圆角：`20px`
- 过渡动画：`0.3s ease`
- 悬停效果：卡片上浮 + 阴影加深

## 四、功能清单

### 4.1 核心功能

| 功能编号 | 功能名称 | 功能描述 |
|----------|----------|----------|
| F01 | 游戏展示 | 以卡片网格形式展示所有游戏，包含封面图、名称、类型标签、个人评分 |
| F02 | 分类筛选 | 提供6个分类标签按钮，点击可筛选对应类型的游戏 |
| F03 | 关键词搜索 | 在搜索框输入关键词，实时搜索匹配游戏名称或描述 |
| F04 | 游戏详情 | 点击游戏卡片弹出详情弹窗，展示完整介绍、游戏截图、个人评价 |
| F05 | 数据可视化 | 使用雷达图展示个人在6个游戏维度的偏好程度 |
| F06 | 响应式布局 | 适配不同屏幕尺寸（桌面端、平板、手机） |

### 4.2 交互规则

| 交互场景 | 触发条件 | 交互效果 |
|----------|----------|----------|
| 分类筛选 | 点击分类标签 | 高亮当前标签，过滤显示对应类型游戏，"全部"显示所有 |
| 关键词搜索 | 在搜索框输入文字 | 实时过滤游戏卡片，匹配游戏名称或描述中包含关键词的条目 |
| 查看详情 | 点击游戏卡片 | 弹出模态弹窗，展示游戏详细信息 |
| 关闭详情 | 点击弹窗关闭按钮/遮罩层 | 关闭弹窗，恢复页面 |
| 卡片悬停 | 鼠标悬停在卡片上 | 卡片轻微上浮，阴影加深，显示"查看详情"提示 |
| 空搜索结果 | 搜索无匹配结果 | 显示"未找到相关游戏"提示信息 |

## 五、数据结构

### 5.1 游戏分类体系

| 分类ID | 分类名称 | 包含游戏 |
|--------|----------|----------|
| arpg | 动作角色扮演类（ARPG） | 空洞骑士、Ori |
| sandbox-rpg | 沙盒生存RPG | 泰拉瑞亚 |
| sandbox-survival | 沙盒生存冒险 | 饥荒 |
| co-op | 多人合作休闲 | 双人成行、胡闹厨房、R.E.P.O |
| competitive | 竞技对抗 | 第五人格、王者荣耀 |
| mmorpg | 开放世界MMORPG | 洛克王国世界 |

### 5.2 游戏数据字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | Number | 游戏唯一标识 |
| name | String | 游戏名称 |
| nameEn | String | 游戏英文名称 |
| category | String | 所属分类ID |
| categoryName | String | 分类中文名称 |
| subCategory | String | 子分类描述 |
| image | String | 封面图路径 |
| rating | Number | 个人评分（1-10） |
| playTime | Number | 游戏时长（小时） |
| platform | String | 游玩平台 |
| description | String | 游戏简介 |
| review | String | 个人评价 |
| tags | Array | 游戏标签 |

### 5.3 数据统计维度

雷达图展示以下6个维度的偏好程度（0-10分）：

| 维度 | 说明 |
|------|------|
| 探索冒险 | 对探索未知世界的喜爱程度 |
| 操作挑战 | 对高难度操作和战斗的偏好 |
| 合作社交 | 对多人合作游戏的喜爱程度 |
| 竞技对抗 | 对竞技类游戏的偏好 |
| 建造创造 | 对沙盒建造类游戏的偏好 |
| 剧情叙事 | 对故事剧情的重视程度 |
